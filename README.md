## ✨ A Full-Stack Platform for Event Management

Built using modern web technologies and teamwork.

---

## 🛠 Tech Stack

- 🎨 Frontend: HTML, CSS, Bootstrap, JavaScript
- ⚙️ Backend: Python, Django
- 🗄️ Database: MySQL

---

## 👋 About the Project

EventSphere is a full-stack event management system that allows users to:

- 📅 Discover events
- 🎟 Register easily
- 📌 Manage bookings

---

## 🎯 Core Features

- 🔐 User Authentication
- 📅 Event Listing
- 🎟 Event Registration
- 📌 My Events Dashboard
- ⚡ API Integration

---

## ⚙️ Setup Instructions

```bash
git clone <repo-link>
cd project-folder

Run Backend

git checkout backend
pip install -r requirements.txt
python manage.py runserver

Run Frontend

git checkout frontend
open index.html

🤝 Collaboration
frontend branch → UI development
backend branch → API development
main → final merged project
```
